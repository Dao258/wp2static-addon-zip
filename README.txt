Adds a Zip archive deployment option for WP2Static.

== Description ==

Generate a Zip archive of your static site for custom deployments.

Adds option to rewrite URLs for offline usage

== Installation ==

Upload this plugin's ZIP to your WordPress plugins page within your dashboard.

Activate the plugin, then navigate to your WP2Static main plugin page to see
 the new deployment option available.
